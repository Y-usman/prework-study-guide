HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The nav element acts as a container for webpage navigation elements/content.
  • The section element acts as a container which groups related content in a webpage together.
  • All elements contain attributes which can be likened to the elements characteristics.
  • The footer element contains content which are usually at the bottom/end of a webpage.

CSS

  • Enter your CSS notes here
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The CSS syntax basic building blocks consist of "the property" and "the value".
  • CSS acts as a language for telling browser engines how to render the html pages.

Git

  • Enter your Git notes here
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull: this copies the modified files from the cloud to your local machine
  • git push: this uploads the added changes from your local machien to the remote repository
  • git clone: copies all files from the remote repository onto your local machine
  • git add: this adds the changes from your modified files in the working directory to the staging area
